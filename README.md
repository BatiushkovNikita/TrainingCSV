Тестовое задание:

Создать простое веб приложение, которое позволит импортировать в базу список контактов из CSV файла, 
а также просматривать список импортированных контактов.

Меню
- Страница для отображения списка
- Страница импорта

Импорт должен производиться из CSV файла, который должен содержать следующую информацию:
- Name
- Surname
- Login
- E-mail
- Phone number

Не допускается возможность системы осуществлять одновременный импорт нескольких документов (от разных пользователей) - т.е  если процесс пошел и идет долго (напр 1 миллиард записей), то второй запуск уже не может вклиниться и должен ждать, пока не закончит работу первый.

Импорт должен работать одновременно в двух режимах
- Создание
- Обновление (Update)

Режим отображения списка контактов должен поддерживать следующие опции:
- Пейджинг (возможно "листать" постранично, если список превышает допустимое для отображения количество строк
- Сортировку по каждой колонке

----------------------------------------------------------------------------------------

Запуск:
- Добавить Maven в переменные среды.
- Настроить доступ к базе (MySQL): 
dao/src/main/resources/dbConfig.properties
dao/src/main/resources/database.sql (USE)
- Запустить run.bat
- Собранный проект: web/target/training-csv.war

Если в добавляемой записи поле Login не является уникальным (уже есть в базе), 
то соответсвующая запись будет обновлена.
